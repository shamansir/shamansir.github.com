True Last.FM Compatibility Calculator

  3. Take mismatches in consideration

Calculates the last.fm users compatibility using not just the matched artists, but the number of songs they've listened.

Instructions

  1. Enter the first user name in the first field. That will be the user “at the left hand” in the comparison.
  2. Select “Compare with all friends” to make all your (or any user, entered in the first field) friends to be “at the right hand” in the comparison, one by one (this process may take a really long time). Or just click the “get friends list” line to load friends list and then select concrete friends to compare with. If you want to compare with someone that is not in this friends list, just enter the second user name instead.
  3. If this checkbox is checked, it means that all of the mismatched artists between two users will be added to (subtracted from, to be truth) the final compatibility value and, may be, this will be the “more true” compatibility value.
  4. Press this button when you have entered both usernames or single username and some friends selected. The calculation process will be shown below. If you've selected more than 4 friends or even all friends, please be ready to wait and to receive an evil e-mail from last.fm administrators. You may cancel the calculation process at any time by clicking “Cancel calculation” — it will finish all of the already started micro-transactions and then stop. Also, this button is useful when the repeated errors are received from the last.fm server.
